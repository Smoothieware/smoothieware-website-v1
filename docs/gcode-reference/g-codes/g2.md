{::nomarkdown}
<gcode>G2</gcode> means "move while activating the tool", in a clockwise arc motion.

This command is equivalent to <gcode>G1</gcode>, except the motion is rotational instead of linear.

<gcode>G2</gcode>'s evil twin is <gcode>G3</gcode>, which is exactly the same, except it is counter-clockwise instead of clockwise.
{:/nomarkdown}

## Format

The command is used as such:

```plaintext
G2 X10 Y20 I30 J40 F100
```

Which means: move to X position 10, Y position 20, while maintaining a constant distance from the point of relative coordinates X 30 and Y 40, at a speed of 100 millimeters/minute.

## Parameters

| Parameter | Usage | Example |
| --------- | ----- | ------- |
| `X` | Move to this position in the X axis | <gcode>G2</gcode> X10 I10 |
| `Y` | Move to this position in the Y axis | <gcode>G2</gcode> Y10 I10 |
| `Z` | Move to this position in the Z axis | <gcode>G2</gcode> Z10 I10 |
| `I` | The point in X space from the current X position to maintain a constant distance from | <gcode>G2</gcode> X10 I10 |
| `J` | The point in Y space from the current Y position to maintain a constant distance from | <gcode>G2</gcode> Y10 J10 |
| `K` | The point in Z space from the current Z position to maintain a constant distance from | <gcode>G2</gcode> Z10 K10 |
| `F` | Move at this speed in millimeters/minute (No F parameter results in using last <gcode>G1</gcode> feedrate) | <gcode>G2</gcode> X10 I10 F100 |
