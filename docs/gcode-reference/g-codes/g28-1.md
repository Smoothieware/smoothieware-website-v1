# <gcode>G28.1</gcode> G-code

<gcode>G28.1</gcode> means "store the origin position that the machine goes to when the <gcode>G28</gcode> command is issued".

This position is stored into memory, but is lost if the machine is reset.

You can make that position permanent by sending the <mcode>M500</mcode> G-code.
{:/nomarkdown}

## Format

The command is used as such:

```
G28.1 X10 Y10
```

Which means: record the origin position as being at the X position 10 and Y position 10

## Parameters

| Parameter | Usage                           | Example    |
| --------- | ------------------------------- | ---------- |
| `X`       | Record origin position for X axis | <gcode>G28.1</gcode> X10 |
| `Y`       | Record origin position for Y axis | <gcode>G28.1</gcode> Y10 |
| `Z`       | Record origin position for Z axis | <gcode>G28.1</gcode> Z10 |

## Friend G-codes

{::nomarkdown}
This G-code sets the origin position that the <gcode>G28</gcode> G-code goes to.

And you can use the <mcode>M500</mcode> G-code to save this position into the [config-override](configuring-smoothie) permanently.
{:/nomarkdown}
