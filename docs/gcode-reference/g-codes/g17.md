# {::nomarkdown}<gcode>G17</gcode>{:/nomarkdown} G-code

{::nomarkdown}
<a href="/images/recovered/plane-change-yz.png">
  <img src="/images/recovered/plane-change-yz.png" alt="Change Planes" style="width: 200px; height: 200px; float: right; margin-left: 1rem;"/>
</a>
{:/nomarkdown}

{::nomarkdown}<gcode>G17</gcode>{:/nomarkdown} means "all commands are now to be interpreted in the XY plane".

This command essentially changes the meaning (or plane) of the `I`, `J`, and `K` parameters of the {::nomarkdown}<gcode>G2</gcode>{:/nomarkdown} and {::nomarkdown}<gcode>G3</gcode>{:/nomarkdown} G-codes.

Smoothie starts in the {::nomarkdown}<gcode>G17</gcode>{:/nomarkdown} mode (the default plane is the XY plane).

However, if you changed the plane with the {::nomarkdown}<gcode>G18</gcode>{:/nomarkdown} or {::nomarkdown}<gcode>G19</gcode>{:/nomarkdown} commands, you can come back to the "normal" plane with this command.

This command is modal, meaning that once it is executed, all Gcodes after it are now in this now "mode", until changed by another mode-changing Gcode of its family.

## Format

The command is used as such:

```
G17
```

Which means: {::nomarkdown}<gcode>G2</gcode>{:/nomarkdown} and {::nomarkdown}<gcode>G3</gcode>{:/nomarkdown} commands are now in the XY plane.

## Parameters

No parameters.
