# <gcode>G20</gcode> G-code

<gcode>G20</gcode> means "all moves starting now have their units specified in inches (imperial units)."

<gcode>G20</gcode>'s evil twin is <gcode>G21</gcode>, which means "all moves starting now have their units specified in millimeters (metric units)," which is the default when Smoothie starts.
{:/nomarkdown}

An inch is 25.4 millimeters.

## Format

The command is used as such:

```plaintext
G20
```

Which means: units are now imperial until another G-code changes it (modal command).

## Parameters

No parameters.
