# {::nomarkdown}<gcode>G20</gcode>{:/nomarkdown} G-code

{::nomarkdown}<gcode>G20</gcode>{:/nomarkdown} means "all moves starting now have their units specified in inches (imperial units)."

{::nomarkdown}<gcode>G20</gcode>{:/nomarkdown}'s evil twin is {::nomarkdown}<gcode>G21</gcode>{:/nomarkdown}, which means "all moves starting now have their units specified in millimeters (metric units)," which is the default when Smoothie starts.

An inch is 25.4 millimeters.

## Format

The command is used as such:

```plaintext
G20
```

Which means: units are now imperial until another G-code changes it (modal command).

## Parameters

No parameters.



## V1 and V2 Compatibility

{::nomarkdown}
<versioned orientation="vertical">
<v1>
{:/nomarkdown}

**V1 Implementation:**

The {::nomarkdown}<gcode>G20</gcode>{:/nomarkdown} command in Smoothieware V1 sets all subsequent motion commands to use imperial units (inches).

- **Source:** `Robot.cpp` line 456
- **Function:** `push_state()` modal command
- **Behavior:** Sets `modal_group_1` mode to imperial
- **Duration:** Persists until {::nomarkdown}<gcode>G21</gcode>{:/nomarkdown} is sent or machine is powered off
- **Default:** V1 starts in {::nomarkdown}<gcode>G21</gcode>{:/nomarkdown} (metric) mode
- **Fully supported** - Fundamental G-code that is always available

{::nomarkdown}
</v1>
<v2>
{:/nomarkdown}

**V2 Implementation:**

The {::nomarkdown}<gcode>G20</gcode>{:/nomarkdown} command in Smoothieware V2 maintains identical behavior to V1.

- **Source:** `Robot.cpp` line 768
- **Function:** `push_state()` modal command (identical to V1)
- **Behavior:** Sets `modal_group_1` mode to imperial
- **Duration:** Persists until {::nomarkdown}<gcode>G21</gcode>{:/nomarkdown} is sent or machine is powered off
- **Default:** V2 starts in {::nomarkdown}<gcode>G21</gcode>{:/nomarkdown} (metric) mode
- **Fully supported** - Identical implementation to V1

{::nomarkdown}
</v2>
</versioned>
{:/nomarkdown}

**Status:** Identical implementation across V1 and V2 - no behavioral differences.



## Related Commands

- {::nomarkdown}<gcode>G21</gcode>{:/nomarkdown} - Set units to metric (millimeters)
- {::nomarkdown}<gcode>G0</gcode>{:/nomarkdown} - Rapid positioning
- {::nomarkdown}<gcode>G1</gcode>{:/nomarkdown} - Linear interpolated motion
