# {::nomarkdown}<gcode>G4</gcode>{:/nomarkdown} G-code

{::nomarkdown}
<a href="/images/waiting-room.png">
  <img src="/images/waiting-room.png" alt="Waiting room" style="width: 250px; height: 250px; float: right; margin-left: 1rem;"/>
</a>
{:/nomarkdown}

{::nomarkdown}<gcode>G4</gcode>{:/nomarkdown} means "pause for a given duration of time".

This is useful for example to wait for spindles to ramp up to speed, or to wait for small layers to cool down on 3D printers.

## Format

{::nomarkdown}
<review id="g4:parameter-format">
<proposal>
{:/nomarkdown}

The command is used with either the `S` parameter (seconds) or `P` parameter:

{::nomarkdown}
<versioned orientation="horizontal">
<v1>
{:/nomarkdown}

**V1: Seconds and Milliseconds**

```plaintext
G4 S5
```

Pause for 5 seconds.

{::nomarkdown}
</v1>
<v2>
{:/nomarkdown}

**V2: Seconds and Milliseconds**

```plaintext
G4 S5
```

Pause for 5 seconds.

{::nomarkdown}
</v2>
</versioned>
{:/nomarkdown}

{::nomarkdown}
</proposal>
<original>
{:/nomarkdown}

The command is used as such:

```plaintext
G4 S5
```

Which means: pause for 5 seconds.

{::nomarkdown}
</original>
</review>
{:/nomarkdown}

## Parameters

{::nomarkdown}
<review id="g4:parameter-table">
<proposal>
{:/nomarkdown}

{::nomarkdown}
<versioned>
<v1>
{:/nomarkdown}

| Parameter | Usage                               | Example  |
|-----------|-------------------------------------|----------|
| `S`       | Duration to wait in seconds         | <gcode>G4</gcode> S10   |
| `P`       | Duration to wait in milliseconds (thousandth of a second) | <gcode>G4</gcode> P100 |

When {::nomarkdown}<setting v1="grbl_mode"></setting>{:/nomarkdown} is enabled, the `P` parameter is interpreted as float seconds (not milliseconds) to conform to standard G-Code specifications.

{::nomarkdown}
</v1>
<v2>
{:/nomarkdown}

| Parameter | Usage                               | Example  |
|-----------|-------------------------------------|----------|
| `S`       | Duration to wait in seconds         | <gcode>G4</gcode> S10   |
| `P`       | Duration to wait in milliseconds (thousandth of a second) | <gcode>G4</gcode> P100 |

When {::nomarkdown}<setting v2="general.grbl_mode"></setting>{:/nomarkdown} is enabled, the `P` parameter is interpreted as float seconds (not milliseconds) to conform to standard G-Code specifications.

{::nomarkdown}
</v2>
</versioned>
{:/nomarkdown}

{::nomarkdown}
</proposal>
<original>
{:/nomarkdown}

| Parameter | Usage                               | Example  |
|-----------|-------------------------------------|----------|
| `S`       | Duration to wait in seconds         | <gcode>G4</gcode> S10   |
| `P`       | Duration to wait in milliseconds (thousandth of a second) | <gcode>G4</gcode> P100 |

Note that if {::nomarkdown}<setting v1="grbl_mode" v2="general.grbl_mode"></setting>{:/nomarkdown} is set to `true` then the `P` parameter is the duration to wait in seconds, not milliseconds, as a float value. This is to conform to G-Code standards.

{::nomarkdown}
</original>
</review>
{:/nomarkdown}
