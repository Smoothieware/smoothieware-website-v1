# {::nomarkdown}<gcode>G19</gcode>{:/nomarkdown} G-code

{::nomarkdown}
<a href="/images/recovered/plane-change-yz.png">
  <img src="/images/recovered/plane-change-yz.png" alt="Change Planes" width="200" height="200" style="float: right; margin-left: 1rem;"/>
</a>
{:/nomarkdown}

{::nomarkdown}<gcode>G19</gcode>{:/nomarkdown} means "all commands are now to be interpreted in the YZ plane".

This command essentially changes the meaning (or plane) of the `I`, `J`, and `K` parameters of the {::nomarkdown}<gcode>G2</gcode>{:/nomarkdown} and {::nomarkdown}<gcode>G3</gcode>{:/nomarkdown} G-codes.

Smoothie starts in the {::nomarkdown}<gcode>G17</gcode>{:/nomarkdown} mode (the default plane is the XY plane).

You can change this with the {::nomarkdown}<gcode>G18</gcode>{:/nomarkdown} or {::nomarkdown}<gcode>G19</gcode>{:/nomarkdown} commands, and you can come back to the "normal" plane with the {::nomarkdown}<gcode>G17</gcode>{:/nomarkdown} command.

This command is modal, meaning that once it is executed, all Gcodes after it are now in this now "mode", until changed by another mode-changing Gcode of its family.

## Format

{::nomarkdown}
<versioned orientation="vertical">
<v1>
{:/nomarkdown}

The command is used as such:

```
G19
```

Which means: {::nomarkdown}<gcode>G2</gcode>{:/nomarkdown} and {::nomarkdown}<gcode>G3</gcode>{:/nomarkdown} commands are now in the YZ plane.

{::nomarkdown}
</v1>
<v2>
{:/nomarkdown}

The command is used as such:

```
G19
```

Which means: {::nomarkdown}<gcode>G2</gcode>{:/nomarkdown} and {::nomarkdown}<gcode>G3</gcode>{:/nomarkdown} arc commands are now interpreted in the YZ plane.

{::nomarkdown}
</v2>
</versioned>
{:/nomarkdown}

## Parameters

No parameters.

## Related Commands

- {::nomarkdown}<gcode>G17</gcode>{:/nomarkdown} - Select XY plane (default)
- {::nomarkdown}<gcode>G18</gcode>{:/nomarkdown} - Select XZ plane
- {::nomarkdown}<gcode>G2</gcode>{:/nomarkdown} - Clockwise arc
- {::nomarkdown}<gcode>G3</gcode>{:/nomarkdown} - Counterclockwise arc
