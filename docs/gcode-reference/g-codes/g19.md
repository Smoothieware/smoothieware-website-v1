# <gcode>G19</gcode> G-code
{:/nomarkdown}

{::nomarkdown}
<a href="/images/recovered/plane-change-yz.png">
  <img src="/images/recovered/plane-change-yz.png" alt="Change Planes" width="200" height="200" style="float: right; margin-left: 1rem;"/>
</a>
{:/nomarkdown}

{::nomarkdown}
<gcode>G19</gcode> means "all commands are now to be interpreted in the YZ plane".

This command essentially changes the meaning (or plane) of the `I`, `J`, and `K` parameters of the <gcode>G2</gcode> and <gcode>G3</gcode> G-codes.

Smoothie starts in the <gcode>G17</gcode> mode (the default plane is the XY plane).

You can change this with the <gcode>G18</gcode> or <gcode>G19</gcode> commands, and you can come back to the "normal" plane with the <gcode>G17</gcode> command.

This command is modal, meaning that once it is executed, all Gcodes after it are now in this now "mode", until changed by another mode-changing Gcode of its family.
{:/nomarkdown}

## Format

The command is used as such:

```
G19
```

{::nomarkdown}
Which means: <gcode>G2</gcode> and <gcode>G3</gcode> commands are now in the YZ plane.
{:/nomarkdown}

## Parameters

No parameters.
