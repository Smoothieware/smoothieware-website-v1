# {::nomarkdown}<gcode>G3</gcode>{:/nomarkdown} G-code

{::nomarkdown}
<a href="/images/gcode.png">
  <img src="/images/gcode.png" alt="G-code icon" width="250" height="250" style="float: right; margin-left: 1rem;"/>
</a>

The {::nomarkdown}<gcode>G3</gcode>{:/nomarkdown} command is exactly the same as the {::nomarkdown}<gcode>G2</gcode>{:/nomarkdown} command, but counter-clockwise instead of clockwise.

See the {::nomarkdown}<gcode>G2</gcode>{:/nomarkdown} page for more information.

{::nomarkdown}
<sl-alert variant="neutral" open>
  <sl-icon slot="icon" name="info-circle"></sl-icon>
  <strong>Related Commands:</strong>
  <br><br>
  <gcode>G3</gcode> is the counter-clockwise version of the <a href="g2"><gcode>G2</gcode></a> (clockwise arc) command. Both commands use the same parameters and syntax, differing only in the direction of the arc movement.
</sl-alert>
{:/nomarkdown}
