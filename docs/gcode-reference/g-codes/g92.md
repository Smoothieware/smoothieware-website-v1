## Description

{::nomarkdown}
The <gcode>G92</gcode> G-code means "set an offset for all coordinate systems".

To see more information on the <gcode>G92</gcode> Gcode see [G92-cnc](g92-cnc).
{:/nomarkdown}

## 3D Printer Usage

{::nomarkdown}
On 3D printers, <gcode>G92</gcode> E0 will reset the currently selected extruder position to 0.
{:/nomarkdown}

This is commonly used at the beginning of print jobs to ensure the extruder position starts from a known state.
