## Description

The {::nomarkdown}<gcode>G92</gcode>{:/nomarkdown} G-code means "set an offset for all coordinate systems".

To see more information on the {::nomarkdown}<gcode>G92</gcode>{:/nomarkdown} Gcode see [G92-cnc](g92-cnc).

## 3D Printer Usage

On 3D printers, {::nomarkdown}<gcode>G92</gcode>{:/nomarkdown} E0 will reset the currently selected extruder position to 0.

This is commonly used at the beginning of print jobs to ensure the extruder position starts from a known state.
