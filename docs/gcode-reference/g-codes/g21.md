## <gcode>G21</gcode> G-code
{:/nomarkdown}

{::nomarkdown}
<a href="/images/mode-switch.png">
  <img src="/images/mode-switch.png" alt="Change Modes" style="width: 200px; height: 200px; float: right; margin-left: 1rem;"/>
</a>
{:/nomarkdown}

<gcode>G21</gcode> means "all moves starting now have their units specified in millimeters (metric units)".

<gcode>G21</gcode>'s evil twin is <gcode>G20</gcode>, which means "all moves starting now have their units specified in inches (imperial units)".

An inch is 25.4 millimeters.

<gcode>G21</gcode> is the default when Smoothie starts.
{:/nomarkdown}

### Format

The command is used as such:

```plaintext
G21
```

Which means: units are now metric until another G-code changes it (modal command).

### Parameters

No parameters.
