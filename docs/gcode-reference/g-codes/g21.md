## {::nomarkdown}<gcode>G21</gcode>{:/nomarkdown} G-code

{::nomarkdown}
<a href="/images/mode-switch.png">
  <img src="/images/mode-switch.png" alt="Change Modes" style="width: 200px; height: 200px; float: right; margin-left: 1rem;"/>
</a>
{:/nomarkdown}

{::nomarkdown}<gcode>G21</gcode>{:/nomarkdown} means "all moves starting now have their units specified in millimeters (metric units)".

{::nomarkdown}<gcode>G21</gcode>{:/nomarkdown}'s evil twin is {::nomarkdown}<gcode>G20</gcode>{:/nomarkdown}, which means "all moves starting now have their units specified in inches (imperial units)".

An inch is 25.4 millimeters.

{::nomarkdown}<gcode>G21</gcode>{:/nomarkdown} is the default when Smoothie starts.

### Format

The command is used as such:

```plaintext
G21
```

Which means: units are now metric until another G-code changes it (modal command).

### Parameters

No parameters.
