{::nomarkdown}
## <mcode>M23</mcode>: Select File

Issuing an <mcode>M23</mcode> selects a file from the SD card that will be used for printing with the <mcode>M24</mcode> command.

The file will remain "selected" and all subsequent <mcode>M24</mcode> commands will print that file until a different file is selected.
{:/nomarkdown}

### Error Handling

If the file does not exist on the SD card, an error will be returned (ERROR?).

### Restarting Files

{::nomarkdown}
The <mcode>M23</mcode> command can be used to restart a file from the beginning.

Use <mcode>M23</mcode> to select/reset the file, then <mcode>M24</mcode> to start printing from the beginning.
{:/nomarkdown}

{::nomarkdown}
<sl-alert variant="neutral" open>
  <sl-icon slot="icon" name="info-circle"></sl-icon>
  <strong>Print:</strong> In this context, "print" refers to running a job on any Smoothie-controlled device such as 3D printer, laser engraver/cutter, CNC router, etc.
</sl-alert>
{:/nomarkdown}

## Format

The command is used as such:

```
M23 filename.gco
```

{::nomarkdown}
Which means: Select the file to be used in the subsequent <mcode>M24</mcode> print/resume command.
{:/nomarkdown}

## Parameters

| Parameter | Usage | Example |
| --------- | ----- | ------- |
| `filename` | The file, on the SD card, will be printed in the subsequent M24 command | M20 myGcodeFile.gco |

### Filename Support

{::nomarkdown}
The `filename` parameter identifies the file to be used by subsequent <mcode>M24</mcode> commands.
{:/nomarkdown}

Smoothie supports both long filenames and [8.3 format](https://en.wikipedia.org/wiki/8.3_filename).

## Example

If your SD card contents were:

- **config.txt**
- **firmware.cur**
- **Readme.txt**
- **myGcode.gco**

This means if you command:

```
M23 myGcode.gco
```

{::nomarkdown}
The file myGcode.gco would be queued for subsequent printing with the <mcode>M24</mcode> command.

To actually start printing, you would then issue:
{:/nomarkdown}

```
M24
```

## Configuration

{::nomarkdown}
The <mcode>M23</mcode> command requires no special configuration settings.
{:/nomarkdown}

The SD card must be properly inserted and contain the file you're trying to select.
