# 论坛讨论：设置 Mini X5 时遇到的问题

## 用户提问
**用户**: Danny D  
**时间**: 2014年7月28日 01:39  
**主题**: 设置 Mini X5 时遇到的问题  

**内容**:  
在使用 Repetier Host 时，发现无法使用两个功能：  
- 调整打印速度的滑块（Feed Rate）  
- 调整挤出速度的滑块（Extrude Rate）  

询问是否有人知道如何解决这个问题。

## 回答
**回答者**: wolfmanjm  
**时间**: 2014年7月31日 07:26  
**主题**: Re: 设置 Mini X5 时遇到的问题  

**内容**:  
由于 Repetier Host 是闭源软件，因此在 Smoothie 中可能不会完全支持，所提到的两个功能可能因此无法使用。  
建议使用开源软件 **Printrun / Pronterface**，这些工具与 Smoothie 兼容性良好，且可以修复潜在的不兼容问题。
