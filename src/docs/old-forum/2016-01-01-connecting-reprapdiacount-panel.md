# Connecting RepRapDiscount Panel Issue

## Original Problem
The user encountered issues connecting their RepRapDiscount Panel. They initially suspected the problem was due to a failed firmware flash. However, the root cause was a configuration error: the required settings were mistakenly pasted near the top of the config file, while the actual settings were set to `false` at the end of the file.

## Resolution
The user identified two key mistakes:
1. **Failed firmware flash**: The initial attempt to flash the newest firmware did not succeed.
2. **Incorrect configuration placement**: The settings for the panel were incorrectly placed near the top of the config file instead of being properly configured at the end.

After correcting the configuration file and re-flashing the firmware, the panel began to work, though it remained slightly flaky. The user noted that the RepRapDiscount Panel, purchased as the cheapest option from eBay, functioned flawlessly aside from their own human errors.

> **Note**: The user humorously mentioned that the resolution allowed them to finally get their laptop out of the garage.
