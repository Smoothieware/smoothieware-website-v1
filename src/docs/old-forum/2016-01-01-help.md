# Help: Trouble Writing Config File

*Author: Helipad14399 | Date: 2016-06-06 16:30*

I'm trying to set up my 3D printer (Smoothieboard) and I'm having trouble writing the config file to the SD card. I've tried using both the SD card that came with the board and a new one, but neither seems to work. The printer doesn't recognize the config file, and I get an error message. I've checked the file format and it's in the correct format (INI). Any ideas on what could be causing this issue?

---

*Author: marcos | Date: 2016-06-06 17:15*

Have you tried formatting the SD card? Sometimes the file system can cause issues with the printer recognizing the files. Also, make sure the config file is named correctly (e.g., `printer.cfg`).

---

*Author: Helipad14399 | Date: 2016-06-07 03:42*

I tried formatting the SD card, but it didn't help. The file is named `printer.cfg` and is in the root directory of the SD card. I'm starting to think it might be a problem with the SD card itself. I'll try using a different one.

---

**UPDATE | It Works!**

*Author: Helipad14399 | Date: 2016-06-15 17:42*

Turns out the SD Card was giving me trouble. Not sure if it's because it had lots of files on it or just that the SD Card itself couldn't keep up (This was the SD Card that came with my Smoothieboard). I went out and bought a Sandisk Ultra Class 10 SD Card. Copied my original config file right over and it works!

If anybody is interested, I can post my config file for reference.

Happy Printing!
