# Getting a Smoothie on a breadboard to access the SD Card

*Original post date: 2016-01-01*

---

킹스턴 클래스 4, 4GB 마이크로 SD 카드를 사용하여 스피드와 SD 카드 라이브러리가 호환되는 문제를 해결했습니다. MOSI, MISO, CE에 6k8 풀업 저항을 사용하여 3V3에 연결했습니다. 이 과정에서 다양한 카드를 테스트했고, 결국 킹스턴 클래스 4 카드가 작동했습니다. 이 정보를 바탕으로 FirePick에서의 회전 델타 코드를 Smoothie로 이전할 수 있게 되었습니다.

---

**요약:**

- **사용한 카드:** Kingston Class 4, 4GB micro SD
- **필요한 회로 구성:** MOSI, MISO, CE에 6k8 풀업 저항 (3V3에 연결)
- **결론:** 특정 브랜드 및 클래스의 카드만 호환됨
- **다음 단계:** FirePick의 회전 델타 코드를 Smoothie로 이전

---

**참고 사항:**

- SPI/SD 카드 라이브러리의 제한성
- 테스트 과정에서 다양한 카드 사용
- 성공적인 작동 후 코드 이전 가능

---

**추가 정보:**

- **킹스턴 클래스 4 카드 특징:** 높은 속도와 안정성
- **풀업 저항 역할:** 신호의 안정성 확보
- **3V3 전압:** 전자 회로에서 일반적으로 사용되는 전압

---

**결론:**

킹스턴 클래스 4 카드와 적절한 회로 구성으로 문제를 해결했으며, 이는 특정 카드와 라이브러리 호환성의 중요성을 보여줍니다. 이후 코드 이전 작업을 통해 프로젝트의 진행이 가능해졌습니다.

### 문제 해결 요약

킹스턴 클래스 4, 4GB 마이크로 SD 카드를 사용하여 SPI 인터페이스를 통해 SD 카드와의 통신 문제를 해결했습니다. 이 과정에서 MOSI, MISO, CE 신호 라인에 6k8 풀업 저항을 3V3 전압에 연결하여 신호의 안정성을 확보했습니다.
