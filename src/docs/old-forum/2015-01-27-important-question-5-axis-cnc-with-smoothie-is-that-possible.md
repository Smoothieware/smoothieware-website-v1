**IMPORTANT Question 5 axis CNC with smoothie, Is that possible?**

Hi,

I'm considering using the Smoothie firmware for a 5-axis CNC project. I have a few questions:

1. Does the Smoothie firmware currently support 5-axis machining?
2. If not, are there any workarounds or alternative configurations that could achieve similar functionality?
3. What are the limitations of using the extruder modules as alternatives for 5-axis control?
4. Are there any community resources or documentation that discuss 5-axis setups with Smoothie?

I'm particularly interested in using the Smoothie board with Nema 24 stepper motors and a 5-axis machine similar to the [5-axis CNC mill](https://www.example.com/5-axis-mill) from Example Manufacturer.

Any insights or experiences with 5-axis configurations using Smoothie would be greatly appreciated!

### Reply by arthurwolf (31 Jan 2015 15:54)
**Re: IMPORTANT Question 5 axis CNC with smoothie, Is that possible?**

Hey there!

Right now Smoothie firmware does not have 5-axis machining support.

You could use the extruder modules as an alternatives, but it wouldn't be as good as native support.
