I am having an issue with the Azteeg X5 Mini V2 endstop. When I try to use the endstop, it only works when both endstops are triggered at the same time. If I trigger one endstop individually, it doesn't work. I have tried different firmware versions and checked the wiring, but the problem persists.

Here are the details:

- **Endstop Type**: Mechanical
- **Wiring**: Connected to the X and Y axes
- **Firmware**: Tried multiple versions
- **Board**: Azteeg X5 Mini V2

Any help or suggestions would be greatly appreciated.
